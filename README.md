# KHU Computer Association Website
A simple PHP MVC website for the university computer science association.

## Installation
### 1. Clone or Download the project
```bash
cd C:/xampp/htdocs
```
```bash
git clone https://github.com/MeHajHo3ein/Scientific-association-site.git
```
```bash
cd Scientific-association-site
```

### 2. Setup the Database
Open your browser and go to phpMyAdmin:
```bash
http://localhost/phpmyadmin
```
Click on the **Import** tab and select the `database.sql`

### 3. Run the project
Start the PHP built-in server with this command:
```bash
php -S localhost:8000 -t public
```
Now open your browser and visit:
```bash
http://localhost:8000
```
